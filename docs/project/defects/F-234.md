## F-234: Only 35/39 DJ links coming up in local-demo (OPEN)

**Filed:** 2026-03-31 (session 6)
**Severity:** Medium
**Status:** Open
**Affects:** Local-demo DJ mode link topology
**Found by:** Owner (session 6, morning testing)

### Description

In local-demo DJ mode, only 35 of the expected 39 links are established.
4 links are missing. The expected count of 39 is for a 2-way speaker
configuration.

### Likely Cause

The missing links are probably optional links that silently fail due to
node readiness timing — a node may not have its ports available when GM
attempts to create the link. GM's reconciler should retry, but if the
node appears after the initial reconciliation pass, the links may not
be created.

### Investigation Needed

- Identify which 4 links are missing (compare `pw-link -l` output
  against GM's expected link set for DJ mode)
- Check if missing links are pcm-bridge monitor links, IEM links,
  or other optional paths
- Verify GM reconciler retry behavior for late-appearing nodes
