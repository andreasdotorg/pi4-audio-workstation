## F-253: Node.js 20 deprecation in actions/checkout@v4

**Filed:** 2026-04-04
**Severity:** Low
**Status:** RESOLVED (fix in CI restructure commit — actions/checkout@v6)
**Affects:** `.github/workflows/ci.yml` (both jobs use `actions/checkout@v4`)
**Related:** CI run 23978434071, GitHub deprecation notice
**Origin:** CI deprecation warning (owner directive: all CI issues tracked)
**Deadline:** 2026-06-02 (GitHub enforced)

### Description

`actions/checkout@v4` uses Node.js 20, which GitHub is deprecating. After
June 2, 2026, workflows using Node.js 20 actions will begin failing or
showing degraded behavior. The CI workflow uses `actions/checkout@v4` in
both the `test-all` and `test-integration-browser` jobs.

### Fix

Upgrade `actions/checkout@v4` to `actions/checkout@v6` (or latest) which
uses Node.js 22+. This is a one-line change per job in `ci.yml`:

```yaml
# Before:
- uses: actions/checkout@v4
# After:
- uses: actions/checkout@v6
```

Also check other actions for Node.js 20 dependencies:
- `cachix/install-nix-action@v31` — verify Node.js version
- `cachix/cachix-action@v17` — verify Node.js version (replaced `nix-community/cache-nix-action@v6`)

### Priority

Low severity but hard deadline. Should be fixed before June 2, 2026.
Can be bundled with any CI workflow update.
