## F-283: test_production.py partially mislabeled -- 14/27 tests API-only

**Filed:** 2026-04-06
**Severity:** Medium (test infrastructure)
**Status:** OPEN
**Affects:** `src/web-ui/tests/e2e/test_production.py`
**Origin:** QE E2E audit (session 14)

### Description

14 of 27 tests in `test_production.py` use direct API calls only -- no browser
interaction. The remaining 13 tests do use browser interaction and are properly
classified as E2E.

### Fix

Reclassify the 14 API-only tests: move to `tests/service-integration/`.
Keep the 13 browser-driven tests in the E2E directory.
