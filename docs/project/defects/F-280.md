## F-280: test_convolver_reload.py mislabeled as E2E -- all 5 tests are pw-cli subprocess tests

**Filed:** 2026-04-06
**Severity:** Medium (test infrastructure)
**Status:** OPEN
**Affects:** `src/web-ui/tests/e2e/test_convolver_reload.py`
**Origin:** QE E2E audit (session 14)

### Description

All 5 tests in `test_convolver_reload.py` are located in the `e2e/` directory
but interact with PipeWire via `pw-cli` subprocess calls. They do not use a
browser or the web UI at all. These are infrastructure-level integration tests,
not E2E tests.

### Fix

Reclassify: move to `tests/service-integration/`.
