## F-232: Topology endpoint returns stale/empty GM data (OPEN)

**Filed:** 2026-03-30 (session 6)
**Severity:** High (core monitoring data unreliable)
**Status:** Open
**Affects:** Web UI graph tab, topology API endpoint, GM state reporting
**Found by:** QE (session 6, exploratory testing of local-demo)

### Description

The topology API endpoint (`/api/v1/topology` or equivalent) returns
stale or empty data from the GraphManager. The web UI graph tab may show
outdated link topology or empty/zero link counts even when GM has
established links and the audio graph is functional.

### Impact

Operator cannot trust the graph visualization or link count display to
reflect the actual PipeWire topology. This undermines the monitoring
capability and makes debugging routing issues difficult.
